README - Dataset Original  

Visão Geral
</br>
Este repositório contém o dataset original bruto, disponibilizado sem alterações estruturais em relação à fonte de origem. Os dados foram preservados em seu formato original para garantir rastreabilidade, reprodutibilidade e transparência no processo de preparação e transformação dos dados.

Este conjunto de dados pode ser utilizado como base para processos de limpeza, tratamento, integração, modelagem dimensional, análise exploratória, desenvolvimento de pipelines de dados e projetos de ciência de dados.

Objetivo

O objetivo deste repositório é disponibilizar os dados em sua forma original, permitindo:

- Limpeza dos dados;
- Tratamento de inconsistências;
- Padronização de variáveis;
- Construção de indicadores;
